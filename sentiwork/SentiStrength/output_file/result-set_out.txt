<p>It is clear that in Italian ""una cifra ridicola"" is typically interpreted as ""a ridicolously small quantity"". Nonetheless it is clear from the context that its meaning, in the movie, is the opposite, i.e. ""a ridicolously excessive quantity"". In linguistics, this is called an <a href=""http://www.treccani.it/enciclopedia/antifrasi_%28Enciclopedia-dell%27Italiano%29/"" rel=""nofollow"">antiphrasis</a> and, as such, it is accepted by default and we do not need any prior example: you can do it if you want to, no need to ask anyone's permission, the only limit is that the receiver of your message must be able to follow you. Exactly as happens for other <a href=""http://www.treccani.it/vocabolario/pauroso_%28Sinonimi-e-Contrari%29"" rel=""nofollow"">words or expressions</a> such as ""terribile"", ""pauroso"", ""allucinante"", ""bestiale"", ""da paura"", ""da urlo"", ""pazzesco"", ""spaventoso"", ""tremendo"", ""mostruoso"", etc. The beauty of an antiphrasis is that it requires an increased attention level on the receiver's side.</p>

<p>It is certainly possible that this kind of antiphrases is particularly popular in American English - so what? <em>An antiphrasis is an antiphrasis is an antiphrasis is an antiphrasis</em> (<a href=""http://en.wikipedia.org/wiki/Rose_is_a_rose_is_a_rose_is_a_rose"" rel=""nofollow"">cit.</a>).</p>

2,-3,"<p>In Italian the <em>s-</em> prefix has two functions:</p>

<ol>
<li><p>denote the opposite action, like in <em>tappare/stappare</em>, <em>mettere/smettere</em></p></li>
<li><p>denote an “intensive” action, like in</p>

<ul>
<li><em>forzare/sforzare</em></li>
<li><em>correre/scorrere</em></li>
<li><em>calciare/scalciare</em></li>
</ul></li>
</ol>

<p>The three “intensified” verbs above denote a slight different action, but nobody would interpret <em>scorrere</em> with the meaning <em>not be running</em>.</p>

<p>For a horse we'd use <em>scalciare</em> rather than <em>calciare</em> (both mean <em>to kick</em>). I see no reason for prohibiting the two usages</p>

<ul>
<li>Cancella quello cha hai scritto (erase what you wrote)</li>
<li>Ho scancellato la lavagna (I erased the blackboard)</li>
</ul>

<p>where the intensive prefix has its function to denote a “personal” action, like for the horse that <em>scalcia</em>.</p>

2,-4,"<p>In addition to the other answers, which clearly and correctly identify the suffix <em>&ndash;poli</em> as meaning <em>city</em>, I would like an interesting historical point. The suffix can be used in two different ways:</p>

<ol>
<li>in common words, like <em>baracc&oacute;poli</em>;</li>
<li>part of proper nouns, like <em>Napoli</em> or <em>Costantinopoli</em>. </li>
</ol>

<p><em>Tangentopoli</em> originally was an example of 2., because it was a nickname for Milan (<em>""City of bribes""</em>). <em>Paperopoli</em> is also another example of use in this sense.</p>

<p>After the corruption scandal became such a huge issue, the meaning of the suffix actually changed, and is now used in a manner similar to the English <em>&ndash;gate</em> suffix, which started with a proper name (<em>Watergate</em>) and was then extended to <a href=""http://en.wikipedia.org/wiki/List_of_scandals_with_%22-gate%22_suffix"">many other scandals</a>.</p>

<p>In this sense, <em>&ndash;poli</em> is used in common words now, an usage which is called <em>accidental usage</em>, but it has lost the meaning of city, just like <em>&ndash;gate</em> has lost any geographical connotation. It just means <em>large scandal</em>.</p>

<p>A funny side note. The proper noun of the game <em>Mon&oacute;poli</em> is pronounced with a second-last vowel accent like a proper city name. This is to imitate the English game name of <em>Monopoly</em>. In reality, the word <em>monop&oacute;lio</em> does not use the suffix <em>&ndash;poli</em> but the suffix <em>&ndash;polio</em> which means <em>vendor</em>. In fact, the old spelling of the plural <em>monop&oacute;li</em> should be <em>monopolii</em>, <em>monopol&#307;</em> or <em>monopol&icirc;</em>.</p>

<p>Normally, in Italian <em>&ndash;poli</em> never takes an accent.</p>

<ul>

<li>Friùli (not Frìuli)</li>
<li>Nùoro (not Nuòro)</li>
